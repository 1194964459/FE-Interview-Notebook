# 看台布局

参考：https://www.doubao.com/thread/w2acda98c3a140c8b

场馆的看台布局图，这张布局图有**首次使用引导提示**、**点击**具体的看台 可以发起到这个看台的导航、双击放大/缩小 布局图、拖拽布局图

<!-- 状态变量： -->
this.scaling：标记当前是否处于双指缩放模式。
this.moving：标记当前是否处于单指移动模式。
this.prevDistance：双指缩放时，记录上一次两指之间的距离，用于计算缩放比例。
this.prevTouchObj：单指移动时，记录上一次触摸点的坐标，用于计算移动位移。
SCALE_FACTOR：缩放系数（控制缩放的灵敏度）。
MAX_SCALE/MIN_SCALE：缩放的最大、最小限制。

<!-- 操作 -->
双指缩放：touchstart（双指）→ touchmove 触发 handleScale 计算缩放 → touchend 重置模式。
单指移动：touchstart（单指）→ touchmove 触发 handleMove 计算位移 → touchend 重置模式。
模式切换：移动时出现双指则切缩放，缩放时只剩单指则切移动，确保交互的连续性。

<!-- 更新 -->
通过 this.setData 更新视图位置（如调整元素的 transform: translate(x, y)）。
