# 类组件的 Render方法原理

返回JSX, 但 JSX 底层是通过 createElement 来构建虚拟 DOM 结构的。

createElement 语法格式为：
```js
React.createElement(
  type,
  [props],  // 可选
  [...children]  // 可选
)
```

* type：必填参数，用于指定创建的虚拟 DOM 节点的类型，有两种形式：
    * 字符串：用于创建原生 DOM 元素
    * React 组件：函数组件/类组件

* props：可选参数，是一个对象。包括：
    * HTML 原生属性（如 className、id、src 等）、
    * 事件处理函数（如 onClick、onChange 等）
    * 以及传递给子组件的自定义属性（props）。
