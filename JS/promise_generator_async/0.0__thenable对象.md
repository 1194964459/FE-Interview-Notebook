# thenable对象

thenable对象不是标准 Promise，但也能被 Promise 系统处理。

thenable 是指 “具有 then 方法的对象或函数”（不一定是通过 new Promise 创建的）。


```js
// 这不是 Promise 实例，但有 then 方法，所以是 thenable
const myThenable = {
    then: function (onFulfilled, onRejected) {
        setTimeout(() => {
            onFulfilled("我是 thenable 的结果");
        }, 1000);
    }
};

Promise.resolve(myThenable).then(result => {
    console.log(result); // 1s后，输出 "我是 thenable 的结果"
});
```
