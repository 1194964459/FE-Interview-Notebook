// 判断2个数字是否相等、包括浮点数

== 与 ===

* 整数：
```js
const a = 123; // Number（安全整数）
const b = 123n; // BigInt
// 正确判断：均为整数 → 转 BigInt 比较
const isEqual = typeof a === "number" && Number.isInteger(a) && a <= Number.MAX_SAFE_INTEGER 
  ? BigInt(a) === b 
  : false;
console.log(isEqual); // true
```

* 浮点数：
如 `0.1+0.2 vs 0.3`，用 “误差阈值（Number.EPSILON）” 判断差值是否小于极小值（避免精度偏差）；
```js
function isFloatEqual(a, b, epsilon = Number.EPSILON) {
  return Math.abs(a - b) < epsilon;
}
console.log(isFloatEqual(0.1 + 0.2, 0.3)); // true（正确）
console.log(isFloatEqual(99.99 * 3, 299.97)); // true（正确）
```
